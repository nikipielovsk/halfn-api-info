# HalfN Allegro API Integration Info

This repository is a public information page for the **HalfN** Allegro REST API integration.

## Application purpose

HalfN is an internal integration used by the owner of the Allegro seller account to manage selected selling operations through the official Allegro REST API.

The integration may be used for:

- reading seller account and offer data,
- preparing and creating offers,
- updating offer content,
- updating prices and stock,
- checking offer status,
- managing seller-side product and offer data.

## Technical context

The integration communicates with the official Allegro REST API using OAuth 2.0 authorization.

The application is used only for the owner's own Allegro selling workflow. It is not a public SaaS product and is not offered to third-party sellers.

## User-Agent

Recommended User-Agent format for this application:
